# Contributing to Epic Tech AI - Cinematic Portal

Thank you for considering contributing to this cinematic AI universe!

## How to Contribute
1. Fork the repo
2. Create a feature branch (`git checkout -b feature/amazing-idea`)
3. Make your changes
4. Ensure `npm run build` succeeds with zero errors
5. Commit with clear messages
6. Push and open a PR

## What We Love
- Cinematic enhancements (better particles, glitch effects, 3D interactions)
- New links to add to the anti-hero archive carousel
- Improvements to the anti-hero chat persona (deeper cyberpunk flavor)
- Performance optimizations for the background canvas
- New docs or screenshots for the README

## Code of Conduct
Be excellent to each other. The anti-hero is watching.
