## 🛠 Tech Stack
- Next.js 16.2.4 (App Router, Turbopack, React 19)
- Tailwind CSS v4 + custom neon cyberpunk styling
- Framer Motion for cinematic animations and 3D tilt
- Lucide React for icons
- Grok 4 via x.ai API (for the anti-hero chat)
- Canvas for the background rain and volumetric effects
- Vercel for instant deploys with environment variables

## Local Run
```bash
npm install
npm run dev
```

Open http://localhost:3000

## Vercel Deployment
1. Import the repo in Vercel
2. Add `XAI_API_KEY` in Environment Variables (for the anti-hero chat to work with Grok 4)
3. Deploy

Any push to `main` triggers a new build.
